add node and npm to you machine

go to the root folder and from your terminal run 'npm i' - this will pull in all of your dependencies

if you want to add a new page/ component then open your terminal and run ng g c 'name-of-page-etc' this will add the necessary files to your project

If you want to edit your project run  'ng serve' this will run a local version of the website on port 4200 ie localhost:4200

If you want to build the project then run 'ng build' this will build an optimized version of the websites and it is these files that you up,oad to server
